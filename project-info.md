# Project SpaceWolf: Информация о проекте

## Обзор проекта
**Project SpaceWolf** — это инициативный проект космического шутера в жанре **on-rails shooter**, вдохновленный классической серией *StarFox*. Проект находится на стадии активной разработки.

### Цель и задачи
- **Цель:** Создание динамичного аркадного симулятора космических сражений.
- **Задачи:**
  - Разработка уникальных механик поведения противников-«головоломок».
  - Дизайн многоуровневых локаций с различными фазами (On-rails и All-range).
  - Создание атмосферного визуального стиля (Sci-Fi, киберпанк).

---

## Игровые особенности

### Противники
Каждый враг требует уникальной тактики:
1. **Корабль «Скат»**: выпускает энергетические волны; уязвим при подготовке залпа.
2. **Тентакль**: бронированный враг, уничтожается только с помощью **Charge Shot**.
3. **Рой (мошки)**: группа мелких врагов; сбрасываются выполнением маневра **Barrel Roll**.
4. **Тандемные двойники**: два истребителя, связанных лучом; ярость при потере напарника.
5. **Бутон-снайпер**: защищенный шар, раскрывающийся только для атаки.

### Уровни
1. **Уровень 2: Космическая станция**
   - Пролет через кладбище кораблей.
   - Защита эвакуационных транспортов.
   - Штурм вражеского флагмана.
2. **Уровень 3: Древний узел связи**
   - Визуальный стиль в духе «Бегущего по лезвию 2049».
   - Сражение с гигантским Механическим Червем.
   - Финальный забег по узким желобам (Trench Runs).

## Процесс разработки (Этапы)
1. **Исследование:** изучение референсов (*StarFox*), определение ключевых механик (Barrel Roll, зоны обстрела).
2. **Дизайн противников:** проектирование врагов как тактических задач.
3. **Проектирование миров:** внедрение режимов On-rails и All-range, проработка визуальных стилей уровней.

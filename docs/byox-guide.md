# Space Invaders from Scratch (C++/OpenGL) - Подробный гайд

## Цель

В этом документе описана последовательная реализация Space Invaders по серии из пяти статей. Реализация выполнена на Windows с CMake, C++ и библиотеками GLFW + GLEW, с CPU-буфером и выводом через OpenGL.

## Требования

- Windows
- g++ (MinGW)
- CMake 3.16+
- Библиотеки:
  - GLFW 3.4 (уже размещена в src)
  - GLEW 2.3.1 (уже размещена в src)

## Сборка и запуск

1. Создайте папку build внутри src.
2. Откройте терминал и выполните:

```
cd practice-project/src
cmake -S . -B build
cmake --build build --config Release
```

3. Запуск:

```
./build/space_invaders.exe
```

После сборки DLL-файлы GLFW и GLEW копируются рядом с exe автоматически.

## Общая архитектура

Игра реализуется в одном файле, чтобы строго следовать структуре туториала. Основные элементы:

- Буфер отрисовки (массив пикселей uint32_t)
- Простые спрайты (bitmap 0/1)
- Игровые структуры: Player, Alien, Bullet, Game
- Рендер через OpenGL текстуру и полноэкранный треугольник
- Фиксированный timestep 60 FPS

## Часть 1. Окно и базовый цикл

Цель: создать окно, инициализировать OpenGL и запустить цикл.

- Инициализация GLFW
- Создание окна 640x480
- Инициализация GLEW
- Базовый цикл: clear → swap → poll
- Включение V-sync

Результат: открывается окно с заголовком Space Invaders.

## Часть 2. CPU-буфер и вывод через OpenGL

Цель: отрисовывать пиксели в CPU-буфере и выводить его на экран.

- Создается Buffer с массивом пикселей
- Пишется шейдерная программа (Vertex + Fragment)
- Буфер отправляется как текстура
- Используется fullscreen-треугольник
- Реализуется функция очистки буфера

Результат: окно отображает буфер как картинку.

## Часть 3. Игровые объекты и анимация

Цель: добавить структуры объектов и спрайты.

- Определяются Sprite, Player, Alien
- Создаются таблицы спрайтов для нескольких типов инопланетян
- Добавляется анимация (переключение кадров)
- Инициализируется сетка 5x11 инопланетян
- Добавляется игрок в нижней части экрана

Результат: игрок и инопланетяне отображаются с анимацией.

## Часть 4. Ввод и коллизии

Цель: сделать игру интерактивной.

- Обработка клавиш: Left/Right/Space/Escape
- Игрок двигается по X, ограничен границами экрана
- Пули создаются при отпускании Space
- Проверка пересечений (AABB)
- Инопланетяне уничтожаются при попадании, показывается спрайт смерти

Результат: игра становится управляемой и в нее можно стрелять.

## Часть 5. Скоринг и UI

Цель: добавить счет и текстовый вывод.

- Реализован 5x7 ASCII шрифт
- Функции вывода текста и чисел в буфер
- Очки начисляются за типы инопланетян
- Выводится строка SCORE, значение очков и CREDIT 00
- Рисуется разделительная линия

Результат: полный UI как в классическом Space Invaders.

## Настройки управления

- Left / Right - движение
- Space - выстрел
- Escape - выход

## Итог

Игра полностью воспроизводит все этапы из серии туториалов:

- Инициализация графики
- CPU-рендеринг
- Спрайты и анимации
- Ввод, пули и коллизии
- Скоринг и интерфейс

Эта реализация является базой для дальнейших улучшений (движение рядов, волны, враги, звук).

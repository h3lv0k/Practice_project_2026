## Введение

Целью проектной практики являлась разработка концепции и дизайна элементов космического шутера «Project SpaceWolf», а также создание веб-сайтов для презентации основного проекта и результатов вариативной части. В задачи практики входило: изучение референсов жанра on-rails shooter (серия StarFox), проектирование уникальных механик поведения противников, дизайн многоуровневых локаций, освоение генератора статических сайтов Hugo, работа с инструментами Git и Markdown, а также реализация классической игры «Space Invaders» на языке C++ с использованием OpenGL.

Практика проходила в период с 03 февраля 2026 г. по 24 мая 2026 г. под руководством куратора от кафедры «Информатика и информационные технологии».

## 1. Общая информация о проекте

- **Куратор проекта —** Камозин Сергей Андреевич.

- **Название проекта:**  
  «Project SpaceWolf» (космический шутер в жанре on-rails shooter).

- **Цели и задачи проекта:**  
  **Цель** — создание динамичного аркадного симулятора космических сражений с проработанным визуальным стилем и уникальными тактическими задачами.  
  **Задачи:**  
  1. Разработка уникальных механик поведения противников-«головоломок».  
  2. Дизайн многоуровневых локаций с различными фазами (On-rails и All-range).  
  3. Создание атмосферного визуального стиля в духе Sci-Fi и киберпанка.  
  4. Интеграция системы тактических способностей игрока (Charge Shot, Barrel Roll).  
  5. Проектирование детализированных сценариев сражений с боссами и крупными объектами.

## 3. Описание задания по проектной практике

Задание состояло из **обязательной** (создание сайта-презентации и ведение документации) и **вариативной** (практическая реализация технологии из списка «Build Your Own X») частей.

**Обязательная часть:**  
- Изучить синтаксис Markdown и вести всю документацию проекта в этом формате.  
- Создать основной статический веб-сайт на базе Hugo, включающий:  
  * домашнюю страницу с аннотацией проекта;  
  * страницу «О проекте» с подробным описанием игровых особенностей;  
  * страницу «Участники» с указанием вклада каждого члена команды;  
  * страницу «Журнал» с записями о прогрессе разработки;  
  * страницу «Ресурсы» со ссылками на использованные материалы и инструменты.  
- Обеспечить публикацию сайтов через GitHub Pages.

**Вариативная часть:**  
- Выбрать проект из списка «Build Your Own X» и реализовать его.  
- Мною был выбран проект: **«Space Invaders from Scratch»** на языке C++.  
- Реализовать низкоуровневый рендеринг через OpenGL без использования готовых игровых движков.  
- Создать отдельный веб-сайт (также на Hugo) для описания процесса реализации вариативной части.

## 4. Описание достигнутых результатов по проектной практике

### 4.1. Веб-сайты (Hugo)
- Созданы два статических сайта: основной (`site/main-site`) и вариативный (`site/var-site`).  
- В качестве основы использован генератор Hugo и тема Ananke.  
- Контент сайтов структурирован согласно заданию (разделы «О проекте», «Участники», «Журнал», «Ресурсы»).  
- Настроена автоматическая публикация через GitHub.

### 4.2. Документация в Markdown и Git
- Создан структурированный репозиторий с файлами: `README.md`, `project-info.md`, `docs/main-report.md`, `docs/variative-report.md`, `docs/byox-guide.md`.  
- Настроена система контроля версий Git, все изменения фиксировались осмысленными коммитами.

### 4.3. Дизайн-проект игры «Project SpaceWolf»
- Разработана концепция тактических противников, каждый из которых требует особого подхода (Корабль «Скат», Тентакль, Рой, Тандемные двойники, Бутон-снайпер).  
- Спроектированы уровни 2 («Космическая станция») и 3 («Древний узел связи»), включающие пролет через кладбище кораблей и сражения с боссами в узких желобах (Trench Runs).  
- Внедрены элементы визуального стиля в духе «Бегущего по лезвию 2049».

### 4.4. Вариативная часть: «Space Invaders»
- Реализован программный рендеринг (CPU-буфер) с выводом через текстуру OpenGL.  
- Написаны шейдеры для отображения игрового поля.  
- Реализована физика столкновений (AABB), система анимации пришельцев и текстовый интерфейс.  
- Использован стек технологий: C++17, CMake, GLFW, GLEW.

### 4.5. Личный вклад и приобретённые навыки
- **Геймдизайн** — научился проектировать врагов как «тактические задачи» и выстраивать геометрию уровней для on-rails шутера.  
- **C++ и OpenGL** — освоил низкоуровневую работу с графикой, управление памятью кадрового буфера и написание шейдеров.  
- **Веб-технологии** — изучил работу с генератором статических сайтов Hugo, настройку тем и структуру шаблонов.  
- **Markdown и Git** — закрепил навыки ведения технической документации и командной работы через Git.
